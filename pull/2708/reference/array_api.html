Array API standard compatibility

Hint

NumPy API Reference: Array API standard compatibility

DPNP's main namespace as well as the dpnp.fft and dpnp.linalg namespaces are compatible with the 2024.12 version of the Python array API standard.

Inspection

DPNP implements the array API inspection utilities. These functions can be accessed via the __array_namespace_info__() function, which returns a namespace containing the inspection utilities.

__array_namespace_info__

Returns a namespace with Array API namespace inspection utilities.
