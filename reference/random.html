Random Sampling (dpnp.random)¶

Simple random data¶

dpnp.random.rand

Create an array of the given shape and populate it with random samples from a uniform distribution over [0, 1).

dpnp.random.randn

If positive int_like arguments are provided, randn generates an array of shape (d0, d1, …, dn), filled with random floats sampled from a univariate “normal” (Gaussian) distribution of mean 0 and variance 1.

dpnp.random.randint

Return random integers from low (inclusive) to high (exclusive).

dpnp.random.random_integers

Random integers between low and high, inclusive.

dpnp.random.random_sample

Return random floats in the half-open interval [0.0, 1.0).

dpnp.random.random

Return random floats in the half-open interval [0.0, 1.0).

dpnp.random.ranf

Return random floats in the half-open interval [0.0, 1.0).

dpnp.random.sample

Return random floats in the half-open interval [0.0, 1.0).
