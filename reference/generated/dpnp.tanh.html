dpnp.tanh

dpnp.tanh(x1)[source]

Compute hyperbolic tangent element-wise.

For full documentation refer to numpy.tanh.

Limitations

Input array is supported as dpnp.ndarray. Input array data types are limited by supported DPNP Data types.

Examples

>>> import numpy
>>> import dpnp as np
>>> x = np.array([-numpy.pi, numpy.pi/2, numpy.pi])
>>> out = np.tanh(x)
>>> [i for i in out]
[-0.996272, 0.917152, 0.996272]
