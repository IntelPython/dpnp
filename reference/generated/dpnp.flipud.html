dpnp.flipud

dpnp.flipud(m)[source]

Reverse the order of elements along axis 0 (up/down).

For a 2-D array, this flips the entries in each column in the up/down direction. Rows are preserved, but appear in a different order than before.

For full documentation refer to numpy.flipud.

Returns:

out – A view of m with the rows reversed.

Return type:

dpnp.ndarray

Limitations

Parameters m is supported either as dpnp.ndarray or dpctl.tensor.usm_ndarray. Input array data types are limited by supported DPNP Data types. Otherwise TypeError exception will be raised.

See also

dpnp.fliplr

Flip array in the left/right direction.

dpnp.flip

Flip array in one or more dimensions.

Examples

>>> import dpnp as np
>>> A = np.diag(np.array([1., 2., 3.]))
>>> A
array([[1.,  0.,  0.],
       [0.,  2.,  0.],
       [0.,  0.,  3.]])
>>> np.flipud(A)
array([[0.,  0.,  3.],
       [0.,  2.,  0.],
       [1.,  0.,  0.]])

>>> A = np.random.randn(2, 3, 5)
>>> np.all(np.flipud(A) == A[::-1, ...])
array(True)

>>> np.flipud(np.array([1, 2]))
array([2, 1])
