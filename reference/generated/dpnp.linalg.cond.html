dpnp.linalg.cond

dpnp.linalg.cond(input, p=None)[source]

Compute the condition number of a matrix.

For full documentation refer to numpy.linalg.cond.

Limitations

Input array is supported as dpnp.ndarray. Parameter p=[None, 1, -1, 2, -2, dpnp.inf, -dpnp.inf, ‘fro’] is supported.

dpnp.norm
