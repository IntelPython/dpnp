dpnp.fft.fft2

dpnp.fft.fft2(x1, s=None, axes=(- 2, - 1), norm=None)[source]

Compute the 2-dimensional discrete Fourier Transform

Multi-dimensional arrays computed as batch of 1-D arrays

Limitations

Parameter norm is unsupported. Parameter x1 supports dpnp.int32, dpnp.int64, dpnp.float32, dpnp.float64 and dpnp.complex128 datatypes only.

For full documentation refer to numpy.fft.fft2.
